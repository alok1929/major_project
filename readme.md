## How to Run This Project

```sh
 ### Make sure to add the OPENAI_API_KEY in the .env file 
 ```

1. **Start the frontend:**
   ```sh
   cd frontend
   npm install
   npm run dev
   ```

2. **Install Python dependencies:**
   ```sh
   pip3 install -r requirements.txt
   ```

3. **Run the backend or test script:**
   ```sh
   python3 test.py
   ```
